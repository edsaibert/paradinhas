# paradinhas
Jogo tipo Monopoly para a disciplina de Paradigmas da Programação

### Uso

<b>1) Baixando dependências (JavaFX)</b>
```bash
make install
```

<b>2) Compilando o programa</b>
```bash
make compile
```

<b>3) Iniciando o programa</b>
```bash
make run
```

<b>4) Limpando os arquivos binários</b>
```bash
make clean
```
